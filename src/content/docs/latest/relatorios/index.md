A central de **Relatórios** oferece análises detalhadas das operações do estabelecimento, com opções de filtro e exportação.

## Relatórios de Vendas

Acesse **Relatórios → Vendas** para visualizar:

### Visão geral
- Vendas por período
- Vendas por produto e categoria
- Vendas por cliente
- Vendas por ponto de venda e operador
- Ticket médio e itens por venda
- Métodos de pagamento utilizados

### Relatórios específicos
| Relatório | Caminho |
|---|---|
| Geral | `Relatórios → Vendas → Geral` |
| Produtos não encontrados | `Relatórios → Vendas → Produtos não encontrados` |
| Venda x financeira | `Relatórios → Vendas → Venda x financeira` |

### Nota fiscal
| Relatório | Caminho |
|---|---|
| Consulta por produto | `Relatórios → Vendas → Nota fiscal → Consulta por produto` |
| Operação | `Relatórios → Vendas → Nota fiscal → Operação` |
| Cliente | `Relatórios → Vendas → Nota fiscal → Cliente` |
| Produto | `Relatórios → Vendas → Nota fiscal → Produto` |
| Evolução | `Relatórios → Vendas → Nota fiscal → Evolução` |
| Partilha de ICMS | `Relatórios → Vendas → Nota fiscal → Partilha de ICMS` |

## Relatórios de Suprimentos

Acesse **Relatórios → Suprimentos** para visualizar:

### Estoque
- Movimentações de estoque por período
- Posição de estoque atual
- Maior circulação de produtos
- Produtos sem circulação
- Produtos abaixo do mínimo
- Visão financeira do estoque
- Consumo de produtos

| Relatório | Caminho |
|---|---|
| Entrada e saída | `Relatórios → Suprimentos → Estoque → Entrada e saída` |
| Saldo | `Relatórios → Suprimentos → Estoque → Saldo` |
| Maior circulação | `Relatórios → Suprimentos → Estoque → Maior circulação` |
| Sem circulação | `Relatórios → Suprimentos → Estoque → Sem circulação` |
| Abaixo do mínimo | `Relatórios → Suprimentos → Estoque → Abaixo do mínimo` |
| Visão financeira | `Relatórios → Suprimentos → Estoque → Visão financeira` |
| Consumo | `Relatórios → Suprimentos → Estoque → Consumo` |

### Nota de entrada
| Relatório | Caminho |
|---|---|
| Operação | `Relatórios → Suprimentos → Nota de entrada → Operação` |
| Fornecedor | `Relatórios → Suprimentos → Nota de entrada → Fornecedor` |
| Produto | `Relatórios → Suprimentos → Nota de entrada → Produto` |
| Evolução | `Relatórios → Suprimentos → Nota de entrada → Evolução` |
| Produto x fornecedor | `Relatórios → Suprimentos → Nota de entrada → Produto x fornecedor` |

### Outros relatórios de suprimentos
- Ordem de compra
- Perda de produto
- Necessidade de compra

| Relatório | Caminho |
|---|---|
| Ordem de compra | `Relatórios → Suprimentos → Ordem de compra` |
| Perda de produto | `Relatórios → Suprimentos → Perda de produto` |
| Necessidade de compra | `Relatórios → Suprimentos → Necessidade de compra` |

## Relatórios Financeiros

Acesse **Relatórios → Financeiro** para visualizar:

### Geral
- Balancete
- Demonstração de resultado do exercício
- Fluxo de caixa

| Relatório | Caminho |
|---|---|
| Balancete | `Relatórios → Financeiro → Geral → Balancete` |
| DRE | `Relatórios → Financeiro → Geral → DRE` |
| Fluxo de caixa | `Relatórios → Financeiro → Geral → Fluxo de caixa` |

### Caixa
| Relatório | Caminho |
|---|---|
| Por categoria | `Relatórios → Financeiro → Caixa → Categoria` |
| Por cliente | `Relatórios → Financeiro → Caixa → Cliente` |

### Contas
| Relatório | Caminho |
|---|---|
| Contas a pagar | `Relatórios → Financeiro → Conta → A pagar` |
| Contas a receber | `Relatórios → Financeiro → Conta → A receber` |
| Recebimento | `Relatórios → Financeiro → Conta → Recebimento` |

## Exportação

Todos os relatórios podem ser exportados em:

- **PDF** — para impressão e arquivamento
- **Excel (.xlsx)** — para análise em planilhas
- **CSV** — para integração com outros sistemas
