---

O modulo de **Extratos Bancarios** permite visualizar e gerenciar os extratos das contas bancarias cadastradas no sistema, facilitando a conciliacao financeira.

## Acessando os Extratos Bancarios

Acesse **Financeiro -> Extratos Bancarios** para visualizar os extratos das contas bancarias.

## Funcionalidades

### Visualizacao de Extratos

A tela de extratos permite:
- Consultar o extrato de cada conta bancaria cadastrada
- Visualizar lancamentos de entrada e saida
- Filtrar por periodo e conta bancaria

### Importacao de Extratos

Para importar um extrato bancario:

1. Acesse **Financeiro -> Extratos Bancarios**
2. Selecione a conta bancaria
3. Faca o upload do arquivo de extrato
4. O sistema processa os lancamentos e exibe para revisao

### Conciliacao

A conciliacao bancaria permite comparar os lancamentos do extrato com os registros financeiros do sistema:
- Identifique lancamentos nao conciliados
- Vincule lancamentos do extrato a contas a pagar ou receber
- Mantenha o saldo bancario sincronizado com o sistema

## Relacao com Contas Bancarias

Os extratos bancarios estao vinculados as contas bancarias cadastradas em **Preferencias -> Contas Bancarias**. E necessario ter ao menos uma conta bancaria ativa para utilizar esta funcionalidade.
