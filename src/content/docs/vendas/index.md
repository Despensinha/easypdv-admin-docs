title: Vendas
description: Visão geral do módulo de vendas do Despensinha ERP.

O módulo de **Vendas** cobre todo o processo comercial: criação de pedidos, emissão de notas fiscais eletrônicas (NF-e e NFC-e), gestão de cupons de desconto, planogramas de vendas, recibos fiscais e inutilização de numerações NFC-e.

| [NFC-e](/vendas/nfce/) | Cupom fiscal eletrônico para vendas no PDV |
| [Cupons](/vendas/cupons/) | Descontos e promoções |
| [Planograma](/vendas/planograma/) | Planogramas de vendas |
