O **Ponto de Venda (PDV)** representa um caixa físico ou virtual onde as vendas são realizadas. Cada PDV pode ter uma série fiscal própria para emissão de NFC-e.

## Campos Principais

| Campo | Descrição |
|-------|-----------|
| Nome | Identificação do PDV |
| Série NFC-e | Série fiscal para emissão de cupons eletrônicos |
| Armazém | Armazém de onde o estoque é baixado |
| Status | Ativo / Inativo |

## Como Cadastrar um PDV

1. Acesse **Cadastros → Ponto de Venda**
2. Clique em **Novo PDV**
3. Informe o nome e a série NFC-e
4. Vincule ao armazém correspondente
5. Salve

## Terminais

Cada PDV pode ter um ou mais **Terminais** (dispositivos físicos) associados. Os terminais são gerenciados em **Cadastros → Terminal**.
