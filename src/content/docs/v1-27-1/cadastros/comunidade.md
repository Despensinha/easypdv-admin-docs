O módulo de **Comunidade** representa a organização ou estabelecimento principal configurado no sistema. Aqui são definidos os dados do negócio usados em documentos fiscais e relatórios.

## Campos Principais

| Campo | Descrição |
|-------|-----------|
| Razão Social | Nome jurídico do estabelecimento |
| CNPJ | Documento fiscal |
| Regime Tributário | Simples Nacional, Lucro Presumido, etc. |
| Endereço | Endereço fiscal do estabelecimento |
| Logotipo | Imagem usada em documentos e relatórios |

## Como Configurar

1. Acesse **Cadastros → Comunidade**
2. Preencha os dados fiscais e de contato
3. Faça upload do logotipo (opcional)
4. Salve as configurações
