O **Inventário** permite realizar a contagem física do estoque e ajustar o saldo do sistema conforme o resultado apurado.

## Como Realizar um Inventário

1. Acesse **Suprimentos → Inventário**
2. Clique em **Novo Inventário**
3. Selecione o **Armazém** a ser contado
4. O sistema gera a lista de produtos com saldo atual
5. Informe a quantidade contada para cada produto
6. Clique em **Finalizar Inventário** — o sistema calcula as divergências e ajusta o estoque

## Verificação de Inventário

Antes de finalizar, revise o relatório de divergências:
- **Positivo**: quantidade contada maior que o saldo do sistema
- **Negativo**: quantidade contada menor que o saldo do sistema

## Inventário em Tarefas

Para estabelecimentos com grande volume, use **Inventário por Tarefas** para dividir a contagem por seção ou responsável, consolidando ao final.
