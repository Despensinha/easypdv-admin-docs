Os **Recibos Fiscais** permitem registrar e gerenciar comprovantes fiscais de vendas no sistema. Esse recurso centraliza a emissao, consulta e controle de recibos associados a operacoes de venda.

## Campos Principais

| Campo | Descrição |
|-------|-----------|
| Número | Identificador do recibo fiscal |
| Cliente | Cliente associado ao recibo |
| Valor | Valor total do recibo |
| Data de Emissão | Data em que o recibo foi emitido |
| Situação | Status do recibo (ex.: emitido, cancelado) |

## Como Criar um Recibo Fiscal

1. Acesse **Vendas → Recibos Fiscais**
2. Clique em **Novo** para abrir o formulário de criação
3. Preencha os dados do recibo: cliente, itens e valores
4. Confira as informações e clique em **Salvar**

## Listagem e Filtros

A listagem de recibos fiscais permite:

- Visualizar todos os recibos emitidos
- Filtrar por período, cliente ou situação
- Acessar os detalhes de cada recibo clicando sobre ele

## Editar e Remover

- Para editar um recibo, clique sobre ele na listagem e altere os dados necessários
- Para remover, utilize a opção **Remover** no menu de ações do recibo

## Ações Disponíveis

| Ação | Descrição |
|------|-----------|
| Criar | Emite um novo recibo fiscal |
| Editar | Altera os dados de um recibo existente |
| Visualizar | Exibe os detalhes completos do recibo |
| Remover | Remove o recibo fiscal do sistema |
